Detailed Description

Programa principal para compactação e descompactação de arquivos usando o algoritmo de Huffman.

O programa oferece ao usuário uma interface de linha de comando para escolher entre os modos de compactação e descompactação. Ele gerencia a abertura/fechamento de arquivos e orquestra todas as chamadas para as funções dos módulos huffman.h e pqueue_heap.h.
