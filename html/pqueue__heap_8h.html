Reorganiza o heap para baixo (Down-Heap) após uma remoção.

Garante que o nó no índice 'idx' (geralmente a raiz após uma remoção) desça para a posição correta, trocando-o com o menor de seus filhos.

Parameters

Reorganiza o heap para cima (Up-Heap) após uma inserção.

Garante que a propriedade do Min-Heap (pai menor que o filho) seja mantida, movendo o nó 'idx' para cima se ele tiver menor frequência que seu pai.

Parameters
pqPonteiro para a fila de prioridade.
idxO índice do nó a ser verificado e movido para cima.
